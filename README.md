# WEB-STUDIO Landing Page

A responsive landing page for a web studio, designed mobile-first.  
Optimized for mobile, tablet, and desktop devices. Built with HTML, CSS, and
vanilla JavaScript.

## Features

- Mobile-first, fully adaptive layout (mobile, tablet, desktop)
- Header in the normal document flow (not fixed)
- Smooth scroll-to-top button
- Footer with social links and subscription form
- Clean and modern design

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/web-studio-landing.git
   Open index.html in your browser.
   ```
